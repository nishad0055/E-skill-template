Flexbox:

1.The CSS Flexbox offers a one-dimensional layout

2.Flexbox offers greater control over alignment and space distribution between items

3.Flex Direction allows developers to align elements vertically or horizontally, which is used when developers create and reverse rows or columns.

Grid:

1. Grid Layout, is a two-dimensional grid-based layout system with rows and columns, making it easier to design web pages without having to use floats and positioning

2.Grid is made for two-dimensional layout while Flexbox is for one. This means Flexbox can work on either row or columns at a time, but Grids can work on both

3.Grid deploys fractional measure units for grid fluidity and auto-keyword functionality to automatically adjust columns or rows.

Boostrap: Boostrap is an open-source framework that is used to design responsive websites better, faster, and easier. It is a beginner-friendly and the most popular open-source framework that includes HTML, CSS, and JavaScript.
Tailwind: Tailwind CSS is a utility-first CSS framework for building custom user interfaces rapidly and efficiently. It is an inline styling used to achieve a sleek interface without writing code for your own CSS

CSS Box-Model: CSS Box-model Talking about design and layout. The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.

HTML Semantic Tag: The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.
